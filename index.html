Voice AI Agent — Local LLM with Real-Time Speech

Sub-1s barge-in · fully offline · no cloud APIs

A fully-local real-time voice assistant — speak to it, it speaks back, no cloud APIs. ALPR — Real-Time License Plate Reader

YOLOv8 + SORT tracking + character-voting OCR

An end-to-end ALPR pipeline for traffic video. YOLOv8 detects vehicles and plates, SORT tracks each vehicle across frames, and PaddleOCR reads the text — stabilised by a confidence-weighted character voting system per vehicle ID that eliminates single-frame OCR flicker. 03

WhatsApp Intent Router — Hinglish NLP Classifier

DistilBERT fine-tune · 99.9% validation accuracy

A DistilBERT fine-tune (67M params, ~3h on CPU) that routes multilingual WhatsApp customer messages — English, Hindi, Hinglish with typos and emojis — into 5 business intents: complaint, confirmation, document, spam, status inquiry. Reached 99.9% accuracy on 4,757 held-out samples and 100% on the hand-curated test set. The deeper lesson: the original 8-class taxonomy plateaued at 50% — collapsing to cleanly-separable 5 classes plus seed augmentation fixed it without hyper-parameter tuning. Trained end-to-end with class-weighted cross-entropy.

05

RC Extractor — Indian Vehicle Documents

Token-pool detectors · auto-approval pipeline

Rule-based extractor for Indian Vehicle Registration Certificates. PaddleOCR feeds an ordered token-pool detector that pulls out registration number, chassis, engine, owner, dates, maker/model and vehicle class. The key trick is detector ordering — running the easiest-to-hardest fields first locks high-confidence tokens so loose engine-number patterns can't steal them downstream. Each document is auto-tagged AUTO_APPROVED or MANUAL_REVIEW. Streamlit UI for live inspection, plus a batch CSV mode for bulk processing.

06

Attention & Engagement Monitor

MediaPipe FaceMesh + OpenCV solvePnP · CPU-only

A YAML-driven offline video pipeline that classifies a person's attention state frame-by-frame — attentive, looking away, looking up, slacking (prolonged head-down). MediaPipe FaceMesh extracts facial landmarks, OpenCV's solvePnP gives 6-point head pose, and temporal logic smooths noise and collapses state intervals. Outputs an annotated video plus a per-session CSV log. Fully tunable via config, no code changes needed.
